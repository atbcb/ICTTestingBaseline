Test: Sync Media Alternative (Prerecorded) (17.G)

Identify Content

Pre-recorded synchronized multimedia that is clearly labeled as a media alternative for text.

Test Instructions

  1. Identify the text for which the media is an alternative.
  2. Play the media and check that the meaningful audible and visual information is available in the text alternative.

Test Results

If any checks fail, the multimedia is not a media alternative for text and tests 17.D and 17.E should be performed.

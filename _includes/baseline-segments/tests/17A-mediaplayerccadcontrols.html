Test: Media Player Controls (17.A)

Identify Content

Media player that displays video with synchronized audio.

Test Instructions

  1. Check that a user control for the selection of captions is provided. [Section 508 503.4]
  2. Check that a user control for the selection of audio descriptions is provided. [Section 508 503.4]

Test Results

If any of the above checks fail, then Baseline Requirement 17.A-MediaPlayerCCADControls fails.

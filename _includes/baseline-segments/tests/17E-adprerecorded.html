Test: Audio Description (Prerecorded) (17.E)

Identify Content

Pre-recorded synchronized multimedia. Do not include media that is clearly labeled as a media alternative for text.

Test Instructions

  1. Enable audio descriptions and play the media; if a separate audio-described file is provided, test that file.
  2. Check that the audio description adequately describes important visual content, including actions, characters, scene changes, and on-screen text. [SC 1.2.5]

Test Results

If any checks fail, Baseline 17.E-ADPrerecorded fails.

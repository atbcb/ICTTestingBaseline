Test: Visual Headings Programmatic (13.B)

Identify Content

Visually apparent headings, which denote sections of content. Note the hierarchy and structure of each heading.

Test Instructions

  1. Check that all visual headings are programmatically determinable and that programmatic heading levels logically match the visual heading presentation [SC 1.3.1].
  2. Verify headings use semantic markup (<h1>–<h6>) or ARIA `role="heading"` with `aria-level` where appropriate.

Test Results

If the above check fails, then Baseline Test 13.B-VisHeadingProg fails.

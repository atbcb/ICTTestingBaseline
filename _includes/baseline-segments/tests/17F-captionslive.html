Test: Captions (Live) (17.F)

Identify Content

Live synchronized multimedia.

Test Instructions

  1. Enable captions for the live session and verify captions are provided. [SC 1.2.4]
  2. Check that provided captions include dialogue and important sounds; compare audio to captions for accuracy and synchronization, allowing for lower real-time accuracy where appropriate.

Test Results

If any checks fail, Baseline Requirement 17.F-CaptionsLive fails.

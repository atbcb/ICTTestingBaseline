Test: Layout tables

  1. Inspect markup to determine whether the table is used for layout/presentation rather than for tabular data.
  2. If table is layout-only, flag as failure for not conveying semantic relations of data.

12.C Test Procedure for Layout Tables

Baseline Test ID: 12.C-LayoutTable

Identify Content

All content/data visually presented in a table that retains any meaningful sequence when linearized.

Test Instructions

  1. Check that the table used purely for layout purposes does not use data table structure elements (e.g., <th>, scope, headers), and that any programmatic roles used are appropriate for layout only. [SC 4.1.2]

Test Results

If any of the above checks fail, then Baseline Test 12.C-LayoutTable fails.

Test: Video-only Alternative (16.B)

Identify Content

Pre-recorded video-only content that is not labeled as a media alternative for text.

Test Instructions

  1. Check that video-only content information is available via a text alternative or an audio track that describes the video. [SC 1.2.1]
  2. View the video while referring to the alternative and confirm it includes the same information (identify characters and actions as needed).

Test Results

If any checks fail, Baseline Test 16.B-VideoOnlyAlt fails.

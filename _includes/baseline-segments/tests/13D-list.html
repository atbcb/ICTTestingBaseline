Test: Visually Apparent Lists (13.D)

Identify Content

Visually apparent lists, which appear as a grouping of items typically one below the other. Exclude navigation menus. Determine the type of list: unordered, ordered, or description list.

Test Instructions

  1. Check that content with the visual appearance of an unordered list is programmatically an unordered list (<ul>) with <li> items. [SC 1.3.1]
  2. Check that content with the visual appearance of an ordered list is programmatically an ordered list (<ol>) with <li> items. [SC 1.3.1]
  3. Check that description-like lists are marked up with <dl>, <dt>, and <dd> as appropriate. [SC 1.3.1]

Test Results

If any of the above checks fail, Baseline Test 13.D-List fails.

Test: Link Purpose (In Context) (14.A)

Identify Content

All links including scripted elements assigned a link role.

Test Instructions

  1. Check that the combination of accessible name and accessible description is not empty. [SC 4.1.2]
  2. Check that the purpose of each link can be determined from the link and programmatically determinable link context [SC 2.4.4]:
    • the link text
    • the link's accessible name and accessible description
    • text in the same paragraph, list, or table cell as the link
    • associated table header cell for links inside tables

Test Results

If any of the above checks fail, then Baseline Test 14.A-LinkPurpose fails.

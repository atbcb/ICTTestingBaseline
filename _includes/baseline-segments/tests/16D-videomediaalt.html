Test: Video Media Alternative (16.D)

Identify Content

Pre-recorded video-only content that is clearly labeled as a media alternative for text.

Test Instructions

  1. Identify the text for which the media is an alternative.
  2. Play the media and verify the meaningful visual information is available in the text alternative.

Test Results

If the checks fail, the media is not a valid media alternative for text and Baseline Test 16.B should be performed.

Test: Programmatic Headings Visual (13.C)

Identify Content

Programmatically determined headings: <h1> to <h6> or ARIA role="heading".

Test Instructions

  1. Check that each programmatically determinable heading is also serving as a visual heading on the page. Content that is not a visual heading cannot have a role of heading. [SC 1.3.1]

Test Results

If the above check fails, then Baseline Test 13.C-ProgHeadingVisual fails.
